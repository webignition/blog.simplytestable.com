Our 27th updates newsletter went out today.

Highlights from this week:

  • Further Reduced Duplicated Data

    Removed non-essential JavaScript static analysis result details, reducing the full database backup size from about 10 gigabytes to about 200 megabytes and reducing the database backup size to about 2% of what it was.

  • Web Client Database Entirely Rebuildable

    The web client at http://gears.simplytestable.com stores for performance reasons a copy of test results that you access through it.

    It can now restore its database from the core application, making a backup of the database non-essential.

  • Working Towards a Stable Backup Strategy

    The high-level backup strategy is ready and changes are being implemented to make it work.

  • Updating the System to be Put Into a Read-only Mode

    Work has started on updating the worker applications to allow them to safely enter a read-only state.

I'm off on vacation to Los Angeles and San Francisco this coming Friday, so no new newsletters for a couple of weeks.

Expect the next newsletter in 3 weeks on February 20.
